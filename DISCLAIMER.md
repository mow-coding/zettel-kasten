# Disclaimer

WOM-kit is open-source tooling for local-first archive and AI-runtime workflows.

It is provided as-is, without warranty.

## Security Responsibility

Security is a shared responsibility. WOM-kit adds dry-run checks, explicit approval paths, and documentation, but it cannot guarantee prevention of prompt injection, malicious external text, unsafe automation, provider compromise, model failure, or operator misconfiguration.

## Human-In-The-Loop Default

Human-in-the-loop operation is the recommended default.

Full-auto / agent-only operation is advanced and experimental. Operators are responsible for agents, models, permissions, providers, automations, and consequences.

Do not use full-auto operation for financial, medical, legal, safety-critical, destructive, or irreversible workflows without independent controls and professional review.
