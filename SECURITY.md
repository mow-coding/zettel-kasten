# Security Policy

This project is designed around private archives, so security mistakes matter.

## Do Not Commit

Never commit:

- provider tokens,
- OAuth credentials,
- private keys,
- password files,
- real source maps,
- real receipts,
- private zets,
- private AI transcripts,
- raw personal files or media.
