# v0.2.4 Documentation Polish Patch

Date: 2026-05-23
Status: pre-release

## Summary

`v0.2.4` rewrites the public entrypoint documentation into a cleaner open-source project format.

The goal is to make the repository easier to understand for both global readers and Korean-speaking early users.

## What Changed

- Rewrote `README.md` as a focused English project overview.
- Added `README.ko.md` as a full Korean project overview.
- Rewrote `UPGRADE.md` as a cleaner English upgrade guide.
- Added `UPGRADE.ko.md` as a Korean upgrade guide.
- Clarified current status, storage model, text provenance, versioning, privacy boundaries, and authorship.

## 한국어 요약

`v0.2.4`는 공개 저장소의 첫인상을 더 전문적으로 보이게 정리한 문서 패치입니다.

영문 README는 글로벌 독자를 위한 정문으로 정리했고, 한국어 README는 한국어 사용자들이 바로 이해할 수 있도록 별도 문서로 분리했습니다.
