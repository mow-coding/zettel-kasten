# v0.2.3 Bilingual Documentation Patch

Date: 2026-05-23
Status: pre-release

## Summary

`v0.2.3` adds Korean explanations alongside the English public documentation.

The public repository still targets a global audience, but Korean readers should also be able to understand the project quickly.

## 한국어 요약

`v0.2.3`은 공개 저장소의 주요 설명에 한국어 병기를 추가한 문서 패치입니다.

이 프로젝트는 전 세계 독자를 겨냥하지만, 창작자와 초기 사용자들이 한국어 사용자일 수 있으므로 한국어 안내도 함께 제공합니다.

## What Changed

- Added Korean summary to `README.md`.
- Added Korean upgrade guidance to `UPGRADE.md`.
- Kept English as the primary public documentation language while adding Korean support.

Compatible with `v0.2.2`.
