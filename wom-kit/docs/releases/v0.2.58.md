# v0.2.58 - ZET Shared Update Record Review Index

Release date: 2026-06-02

v0.2.58 adds a read-only review index for local ZET shared update record JSON files.

## Added

- CLI `archive shared-update-record-review-index <archive-root> --records-dir <path> --dry-run --format json`.
- MCP `zet_shared_update_record_review_index`.
- Service `zet_shared_update_record_review_index`.
- Documentation: [ZET Shared Update Record Review Index](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/docs/zet-shared-update-record-review-index.md).
- Work log: [Shared Update Review Index Work Log](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/plans/work-log-2026-06-02-shared-update-review-index.md).

## Behavior

- The index is dry-run only.
- It writes no files and returns `would_change: []`.
- It scans only direct-child `.json` files under an archive-relative directory.
- It ignores non-JSON files and does not recurse.
- It sorts records deterministically by archive-relative path.
- It reuses the v0.2.56 single-record review policy instead of maintaining a separate policy list.
- It returns compact per-record summaries without body text, local absolute paths, provider URLs, tokens, secrets, or private source locations.

## Safety Boundary

This release does not implement shared-update review writes, shared-update transport, real ZET transport, receiver-side renewal writes, neighbor feed updates, trust/import/acceptance/anchor, attestation/signature writes, provider sync, WordPress publishing, projection writes or receipts, workers, payments, staking, consensus, blockchain, model training, backpropagation, or full-auto behavior.

MCP exposes only the read-only dry-run tool. It does not expose shared update index apply/write/publish/transport/import/trust/attest/sign/anchor tools.

## Upgrade

No private archive migration is required.

See the [Upgrade Guide](https://github.com/mow-coding/zettel-kasten/blob/main/UPGRADE.md) and [Changelog](https://github.com/mow-coding/zettel-kasten/blob/main/CHANGELOG.md).
