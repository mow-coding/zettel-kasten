# v0.3.38 - Connected Accounts Bridge

Date: 2026-06-15

v0.3.38 adds a read-only connected accounts overview.

## Added

- CLI `archive connected-accounts <archive-root> --dry-run`.
- Aliases:
  - `archive accounts`
  - `archive account-status`
- New document:
  - `wom-kit/docs/connected-accounts.md`

## Behavior

The command reads metadata from:

- `provider-bindings.yml`,
- `source-bindings.yml`,
- ignored local `profiles/local/credential-refs.local.yml`.

It groups provider bindings, IMAP mailbox source accounts, credential catalog
metadata, and credential store-type summaries.

## Safety

The command is read-only. It:

- writes no files,
- reads no secret values,
- reads no environment variables,
- opens no password manager or keyring,
- opens no browser password store,
- starts no OAuth,
- calls no providers,
- opens no IMAP connection,
- reads no mail headers, bodies, attachments, or source bytes,
- echoes no exact credential refs, local paths, usernames, emails, tokens,
  provider URLs, or secret values.
