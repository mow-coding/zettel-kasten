# v0.3.57 - Version Truth Source

Released: 2026-06-16

This release adds a read-only version truth-source checkpoint for WOM-kit.

## Added

- `archive --version` for the running CLI version.
- `archive version [inspection-root] --format text|json` for structured version
  checks.
- `runtime-context` field `wom_kit_version` so AI runtimes can see the running
  kit version while checking archive identity and policy.
- Optional comparison with project pins such as
  `.zettel-kasten/source/installed-version.txt`.
- Documentation and tests for the version truth-source flow.

## Safety Boundary

The new checks write no files, call no providers, read no secrets, and redact
local absolute paths by default.

## Still Future Work

This release does not add automatic upgrade, installer repair, live IMAP
execution, provider sync, credential retrieval, or migration.
