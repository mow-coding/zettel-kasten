# v0.2.57 Release Notes

Date: 2026-06-02

## Summary

v0.2.57 is a readability and release-hygiene patch.

It adds a public capability matrix so readers can distinguish implemented local commands, read-only previews, approval-gated writes, documented-only concepts, local hygiene tools, and work that is not implemented.
