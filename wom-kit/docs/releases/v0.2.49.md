# v0.2.49 Release Notes

Date: 2026-05-27

## Summary

v0.2.49 adds public release link hygiene guardrails.

This is a validation, documentation, and version bookkeeping release. It does not add product CLI/MCP behavior.

## Added

- Local public-link checker at `wom-kit/tools/check_public_links.py`.
- Unit tests for release-note link hygiene and GitHub `blob` URL mapping.
- [Public Release Link Hygiene](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/docs/public-release-link-hygiene.md) documentation.
- Public-safe work log for the v0.2.49 link hygiene batch.

## Fixed

- Converted release note links that were safe in repo-local Markdown but unsafe when copied into GitHub Release bodies.
- Added guardrails against suspicious GitHub file links that use `/tree/` instead of `/blob/`.

## Compatibility

No private archive migration is required.

No archive product CLI or MCP behavior changed.

## Safety Boundary

v0.2.49 does not edit GitHub Releases, fetch external URLs, call providers, publish to WordPress, write projection records, write receipts, run ZET transport, fetch or rank recommendations, update neighbor feeds, create trust/import/acceptance/attestation/signature/minting changes, add background workers, train models, run payments/staking/consensus/blockchain, or enable full-auto behavior.
