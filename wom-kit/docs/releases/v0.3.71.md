# v0.3.71 - IMAP Material Capture Approval Receipt

Status: v0.3.71 approval-gated IMAP material capture approval receipt checkpoint
Date: 2026-06-16

v0.3.71 adds the local approval receipt gate after the IMAP material capture
execution contract is ready and before any future command reads selected
message bodies, attachment bytes, or mail-derived text.

## Added

- `archive imap-mailbox-material-capture-approval-plan <archive-root>
  --material-selection-receipt <archive-relative-json> --capture-action
  message_body_capture|attachment_capture|derived_text_capture --decision
  needs_review|approve_once|deny --dry-run|--approve --format json`.
- Aliases:
  - `archive imap-material-capture-approval-plan`
  - `archive mailbox-material-capture-approval-plan`
  - `archive imap-mailbox-material-capture-approval`
- A public documentation page:
  - [IMAP Mailbox Material Capture Approval Plan](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/docs/imap-mailbox-material-capture-approval-plan.md)

## Behavior

The command validates one non-secret material selection receipt through the
existing material capture execution contract, then previews or writes one
non-secret approval receipt under `receipts/imap/material-capture-approvals/`.

The receipt records a human decision, capture action, selected candidate
indexes, selection receipt SHA-256, and future-adapter contract summary without
recording message material or private locator values.

## Safety Boundary

This release still does not implement message body capture, attachment capture,
or mail-derived text extraction.

The command reads no original execution receipt, opens no IMAP connection, reads
no environment variables, opens no keyring/password manager, reads no message
headers again, reads no message bodies, reads no attachments, creates no
derived text, writes no message material, calls no providers, and echoes no
material selection receipt path, execution receipt path, candidate refs,
subjects, senders, recipients, local paths, tokens, or secret values.
