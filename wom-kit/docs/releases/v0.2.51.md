# v0.2.51 Release Notes

Date: 2026-05-27

## Summary

v0.2.51 adds a local Korean product-language hygiene checker for public documentation.

This is a docs/tooling guardrail and version bookkeeping release. It does not add product CLI/MCP behavior.

- [Korean Product Language Hygiene](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/docs/korean-product-language-hygiene.md) documentation.

- [Public-safe work log](https://github.com/mow-coding/zettel-kasten/blob/main/wom-kit/plans/work-log-2026-05-27-korean-product-language-hygiene-checker.md) for the v0.2.51 checker batch.

## Compatibility

No private archive migration is required.

No archive product CLI or MCP behavior changed.

The checker reads local repository Markdown files only. It does not rewrite files, rename implementation identifiers, fetch external URLs, call providers, or edit GitHub Releases.
