Date: 2026-05-23

## Summary

`v0.2.5` makes the public repository clearer as a showcase and distribution repository.

It adds a public documentation map and separates the project records into:

```text
product blueprint / design philosophy
implementation reference research
implementation plans
work logs
```

- Linked README files to the documentation map and philosophy documents.

## Philosophy Coverage

This release makes the following ideas explicit in public docs:

- human data primitives: text/language, sound, image,
- why original source objects and minted zets must remain separate,
- why original editable text and OCR/AI-derived text have different provenance,
- why `zettel-kasten` matters in AX / AI Transformation,
- how private minted zets can later project into Web3-like sharing,
- how 1:1, 1:many, and many:many `zet` sharing map to messenger, SNS/feed, and collaboration workspace.
