## Clarified

- `zettel-kasten` is memory infrastructure, not just a note app.
- `zet` is always text.
- Minting means private archive issuance, not posting or sharing.
- The base archive model is source/original data + metadata + minted zets + receipts.
- Human data primitives are text/language, sound, and image.
- Original editable text and OCR/AI-derived text have different authority levels.
- HITL and AI-agent harness modes are supported by the same authority model.
- Full-authority agent operation must remain scoped, receipted, auditable, and revocable.
- Object storage includes original documents as well as media.
- Provider-aware archives should handle local folders, Notion, Google Drive, GitHub, object storage, and external URLs with provenance.
- `zet` sharing can project into messenger, SNS/feed, or collaboration workspace behavior.
- The Web3-like property is subject-owned, portable, verifiable memory, not token hype.
