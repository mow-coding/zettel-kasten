# v0.3.53 - IMAP Mailbox Adapter Manifest Write

Date: 2026-06-16

v0.3.53 adds an approval-gated local write command for the non-secret IMAP
mailbox adapter manifest.

The release still does not connect to mail providers. It only lets a human
record the reviewed adapter declaration inside the local archive.

## Added

- CLI `archive imap-mailbox-adapter-manifest-write --dry-run`.
- CLI `archive imap-mailbox-adapter-manifest-write --approve --reviewed-by <actor>`.
- Alias `archive mailbox-adapter-manifest-write`.
- A non-secret manifest file under `config/imap-adapters/`.
- A non-secret write receipt under `receipts/imap/adapter-manifests/`.
- Documentation at `wom-kit/docs/imap-mailbox-adapter-manifest-write.md`.

## Verify

```bash
archive imap-mailbox-adapter-manifest-write <archive-root> \
  --adapter-id local-imap \
  --provider gmail \
  --operation header_metadata_scan \
  --selection-rule newest_first \
  --reviewed-by person:me \
  --dry-run \
  --format json
```

The JSON should include:

```text
lifecycle_action: imap_mailbox_adapter_manifest_write_plan
schema_validation.ok: true
files_written: []
```

Approved local writes require:

```text
--approve --reviewed-by <actor>
```

## Safety

The command remains local and non-secret. It:

- exposes no MCP live write tool,
- writes no files unless `--approve` is used,
- refuses to overwrite an existing manifest or receipt,
- opens no IMAP connection,
- attempts no login,
- selects no mailbox,
- searches no mailbox,
- lists no candidate messages,
- reads no IMAP UIDs or Message-ID values,
- reads no headers, bodies, or attachments,
- creates no derived text,
- retrieves no secrets,
- starts no OAuth,
- calls no providers,
- echoes no email addresses, username values, exact account refs, exact
  credential refs, exact mailbox refs, IMAP host values, provider URLs, message
  ids, subjects, sender or recipient values, attachment names, approval receipt
  paths, selection receipt paths, local absolute paths, tokens, or secret values.

No archive migration is required.
