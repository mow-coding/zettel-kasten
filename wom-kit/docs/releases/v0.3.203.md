# v0.3.203 - Reviewed operator vocabulary correction

v0.3.203 is a user-reviewed correction release for the WOM operator vocabulary
surface introduced in v0.3.202. It is additive and requires no archive migration.

## What changed

- `archive ai-response-concept-guide --topic operator_vocabulary --locale ko-KR`
  now separates reviewed terms into:
  - `confirmed_product_language`
  - `confirmed_operator_language`
  - `needs_user_translation` for future naming-review items
- The reviewed operator-language terms include:
  - `object_id` -> `오브제 아이디`
  - `doctor` -> `검진`
  - `provider` -> `외부 서비스`
  - `containment` -> `포함 관계`
  - `safe_preview` -> `미리보기`
  - `approved_write` -> `승인 후 쓰기`
  - `external_report` -> `공개용 문서`
  - `private_working_note` -> `비공개 문서`
- Core product terms remain unchanged:
  - `edge` -> `엣지`
  - `tie` -> `타이`
  - `node` -> `노드`
  - `zet` means the actual WOM record
  - `zettel` means the abstract zettelkasten idea

## Safety

This is a read-only language-guide correction. It writes no archive files, reads
no archive bodies, calls no providers, reads no secrets, and changes no JSON
field names, receipt schema names, CLI command names, or archive migrations.
