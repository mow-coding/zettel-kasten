# v0.3.202 - WOM operator vocabulary translation

v0.3.202 adds a broader vocabulary layer for AI operators helping a human use
WOM. The layer separates user-confirmed product terms from functional words that
still need user translation review.

No archive migration is required.

## What Changed

`archive ai-response-concept-guide` now supports:

```text
archive ai-response-concept-guide <archive-root> --topic operator_vocabulary --locale ko-KR --dry-run --format json
```

The `all` topic also includes the new `operator_vocabulary` section.

- `WOM`, read as "옴";

## Safety

This is a read-only language guide. It writes nothing, reads no archive bodies,
calls no providers, reads no secrets, and does not change machine, JSON, or
receipt terminology. The AI must not invent Korean product terms for
future `needs_user_translation` entries.
