# v0.3.245 - Top-Level Installation On-Ramp

v0.3.245 makes the already-shipped WOM-kit installation and Agent Skill
lifecycle visible from the repository's first public screen.

## What Changed

- Added a bounded Quick Start to the root English and Korean READMEs.
- Added the exact v0.3.245 GitHub wheel and isolated `uv tool install` command.
- Added an immediate `archive --version` verification step.
- Stated plainly that WOM-kit is not yet published to PyPI and that plain
  `pip` requires a dedicated virtual environment plus the exact wheel.
- Separated Python tool installation from AI-host skill activation.
- Added one read-only `runtime-skill-install --dry-run` activation entry.
- Linked directly to the complete English/Korean Python and Agent Skill guides.

## Safety Boundary

- README text runs no command.
- Wheel installation reads or writes no archive and does not alter AI-host
  configuration.
- Skill activation still requires a separate preview, reviewer, exact plan
  digest, and approval.
- No PyPI publication, automatic install, provider call, MCP write, generated
  graph authority, UI, or archive migration is added.

## Verification

Release verification covers the full WOM-kit test suite, current-version and
README documentation checks, deterministic package resources, a clean wheel
install, Agent Skill install/status/uninstall lifecycle, public link hygiene,
Korean product-language hygiene, public privacy hygiene, and release readiness.
