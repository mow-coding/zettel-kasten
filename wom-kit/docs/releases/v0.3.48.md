# v0.3.48 - KeePassXC Bulk Migration Walkthrough

Date: 2026-06-15

v0.3.48 fills the beginner guidance gap between a credential migration plan
and the actual KeePassXC CSV import screens.

## Added

- `archive beginner-setup-manual --topic credential_bulk_migration --dry-run`.
- KeePassXC 2.7.x CSV bulk migration guidance for:
  - normal import vs passkey import,
  - temporary import database creation,
  - UTF-8 CSV import settings,
  - first-row/header, field separator, text qualifier, and comment character
    choices,
  - Group / Title / Username / Password / URL / Notes column mapping,
  - `Database > Merge from Database`,
  - expected root/group tree shapes,
  - slash-created nested groups,
  - cleanup after the main vault is saved and verified.

## Safety

The manual remains read-only. It:

- reads no CSV,
- opens no KeePassXC window,
- creates no temporary database,
- imports or merges no vault,
- records no database path,
- deletes no temporary files,
- writes no files,
- echoes no secret values, usernames, emails, provider URLs, local paths, CSV
  paths, or vault paths.

No archive migration is required.
