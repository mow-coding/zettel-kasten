# v0.2.0 Public Showcase Baseline

Date: 2026-05-23
Status: pre-release

## Summary

`v0.2.0` is the first public compatibility checkpoint for the `zettel-kasten` and future `zet` project.

It gives outside readers a stable place to inspect the early system concept, specs, schemas, fake examples, and reference implementation.

## Included

- local-first archive protocol documents,
- zettel and zettel-kasten specs,
- JSON schemas,
- fake sample archive,
- early Python CLI tooling,
- early MCP tooling,
- setup and security docs,
- product blueprint for `zettel-kasten` and `zet`,
- versioning and compatibility policy,
- source object storage policy for both document files and media files.

## Upgrade Notes

This is the first public baseline.

There is no public prior version to migrate from.

## Archive Migration

No real private archive migration should be attempted from this release alone.

This release is for review, testing, and protocol discussion.

## Verification

Expected local checks:

```bash
cd wom-kit
python -m unittest discover -s tests
python cli/archive.py doctor examples/fake-life-archive --strict
```

Expected result:

```text
tests pass
doctor reports 0 errors and 0 warnings
```

## Privacy

Do not put real private archives, provider tokens, real source maps, real receipts, or private AI conversations into the public repository.
