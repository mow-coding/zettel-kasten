# ZET Shared Update Record Example

Status: sanitized non-executable example
Version: v0.2.55

This folder contains a public-safe example of a future ZET shared update record.

Files:

- `shared-update.example.json`

The example describes a possible receiver-side review record before local renewal. It does not contain a real zet body, real neighbor identity, real provider URL, real transport endpoint, token, private key, local path, or private archive data.

All mutation, write, transport, provider, trust, import, acceptance, anchor, attestation, signature, and full-auto flags stay `false`.

This example does not implement ZET sharing. It is only a shape reference for future design work.
