# ZET Publication Surface Examples

These examples are public-safe placeholders for v0.2.45.

They show how a future publication surface projection might be described without publishing anything. WordPress is used only as a familiar example of a user-selected surface.

Included files:

- `zet-publication-envelope.example.json`
- `wordpress-title.example.txt`
- `wordpress-post.safe-html.example.html`

Safety boundaries:

- no provider API calls,
- no WordPress publishing,
- no real site URL,
- no token, cookie, password, private key, or credential,
- no local absolute path,
- no private source filename,
- no raw AI conversation,
- no projection receipt,
- no mint, trust, import, acceptance, attestation, signature, anchor, or ZET transport.

Posting is not minting. The canonical zet identity stays in the archive.
