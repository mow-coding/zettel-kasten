# ZET Closed Sharing Examples

These files are sanitized, non-executable examples for v0.2.47.

They show a possible future shape for a closed sharing update, but they are not transport messages, proofs, acceptances, attestations, signatures, projection receipts, publications, or provider payloads.

The examples use placeholder identities only:

- `archive:example:alice`
- `archive:example:neighbor`
- `zet_20260526_closed_sharing_example`
- `surface:example:private-archive`

No real provider URL, token, local path, private source material, raw conversation, or personal data is included.
